## üìã About

<img align="right" width="350" src="https://user-images.githubusercontent.com/74038190/229223263-cf2e4b07-2615-4f87-9c38-e37600f8381a.gif" alt="Coding">

Smart Drive is a **comprehensive management system** designed for driving schools. It streamlines administrative tasks, tracks student progress, manages schedules, and integrates Arduino hardware for real-time vehicle monitoring.

**Why Smart Drive?**

Traditional driving schools rely on paper or basic spreadsheets. Smart Drive brings everything into one powerful desktop application with real-time tracking and automation.

### Prerequisites
Qt 6.x or higher

C++ Compiler (GCC/MinGW/MSVC)

Oracle Database & SQL Developer

Arduino IDE (optional for hardware)

text

### Installation

Clone the repository
git clone https://github.com/adem-chikhaoui/smart-drive-management-software.git
cd smart-drive-management-software

Build the project
